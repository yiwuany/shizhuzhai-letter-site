Seal font assets
================

`ebas927.ttf` is the CNS11643 Shuowen Jiezi TrueType seal script font, packaged by GNU Guix as `font-cns11643-swjz`.

Guix describes the font as a seal script font based on the ancient Shuowen Jiezi text published by Taiwan's Executive Yuan, with 6721 glyphs mapped to modern Unicode variants. Guix lists the license as the CNS11643 non-copyleft license.
